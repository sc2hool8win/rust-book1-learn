### my_package

Section 7-1 で作成した、複数の binary crate を含むパッケージです。

Section 7-1 から Section 7-3 にかけて作成した、本棚ツールの library crate です。

### use_remote_crate

Section 7-3 で作成した、my_library をリモートリポジトリ経由で使用する binary crate です。
